
# Godot in Pears


### 1. Download godot
Download Godot [4.3](https://godotengine.org/article/dev-snapshot-godot-4-3-beta-2/#downloads)
and place executable in root folder.

### 2. Release

[b]uild + [p]atch + [r]elease + [s]eed

```bash
./release.sh bprs dev
```

Or just play locally

```
./release.sh bpd dev
```
